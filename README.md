# Get Covid-19 tweet

---

## How to run 

### set API key

set Twitter API environment for .env

- CONSUMER_KEY
- CONSUMER_SECRET
- ACCESS_KEY
- ACCESS_SECRET

### package install

```
$ pip3 install
```

### run

```
$ python3 main.py
```

### checking duplicate data

```
$ python3 check.py <ARG1: filename>
```
